

## 앱개요

- 앱에 처음 들어가면 랜덤으로 명언 한 개가 출력된다.
- 전체 명언 보기를 클릭하면 저장된 전체 명언들이 listview형태로 출력된다.


</br>

## 적용된 이론

- Lecutre_Wise_Saying에서 배웠던 
외부 폰트 적용, ListView 개념을 이용해서 만들었다.
- dataBinding 개념 또한 등장한다.
